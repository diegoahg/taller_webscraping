# taller_webscraping
Condigo con Pandas
## Pre-Requisitos
- Tener instalado python version 3
## Instalación
- pip install pandas
- pip install lxml

## Quick start
python main.py

## Quick start with Docker
- docker build -t tallerws .
- docker run --rm -it tallerws:latest
